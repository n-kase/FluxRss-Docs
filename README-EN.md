# FluxRss Documentation

FluxRss is an RSS reader for iOS that integrates multiple sources, including local RSS feeds, FreshRSS-compatible services, Readwise Reader, and Instapaper. It offers fast feed updates, flexible filtering, AI-powered summaries, and iCloud sync to streamline daily information management.

---

## Features

### RSS Feed Management
- Add, edit, and delete feeds  
- Register local RSS feeds via URL  
- Organize feeds by category  
- Sort and reorder feeds  
- Import OPML files  
- Automatic favicon retrieval  

### External Service Integration
- **FreshRSS (Google Reader API compatible)**  
  - Authentication  
  - Read/unread sync  
  - Starred item sync  
- **Readwise Reader**  
  - API token authentication  
  - Save articles  
- **Instapaper**  
  - Username/password authentication  
  - Save articles  

### Article Viewing & Management
- Category- and feed-based article lists  
- Article detail view with WebView  
- Read/unread management  
- Star (favorite) support  
- Image preview  
- Embedded video support  
- Filters: unread only, starred only, by category, by feed

## Privacy

FluxRss does not collect personal data.  
API keys and authentication credentials are securely stored in the device Keychain and are never transmitted externally.
